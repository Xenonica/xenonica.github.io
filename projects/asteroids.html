Asteroids (2026)

Boot.dev Python OOP project

Project Overview

This project is a hands-on exercise in object-oriented programming with Python. It applies core OOP concepts — inheritance, polymorphism, encapsulation, and composition — by building a classic arcade game.

The goal is to reinforce OOP principles through practical implementation, creating a fun and interactive game while learning how to structure code effectively.

My Contribution

  • Designed and implemented the game architecture using OOP principles.
  • Created classes for game entities such as Player, Asteroid, and Bullet, utilizing inheritance and polymorphism.
  • Implemented game mechanics including movement, collision detection.
  • Used encapsulation to manage game state and ensure code modularity.
  • Tested and debugged the game to ensure smooth gameplay experience.
