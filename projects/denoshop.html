Project Description

Denoshop is an e-commerce website that allows YouTubers to sell merch to their fans.

We use the Scrum model, the agile approach to organise, plan and provide feedback to each other for this project. Every 1-2 weeks we will meet and discuss together what each of us has done during the past 1-2 weeks related to the project. This sprint cycle continues till the end of the project.

My contribution to the team
  • Implement product management in admin dashboard
  • Implement category management in admin dashboard
  • Implement coupon management in admin dashboard
  • Implement wishlist feature for front-end users
  • Implement pagination on product tab for front-end users
  • Implement product recommendation using Recombee API for front-end users
