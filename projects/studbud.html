StudBud (2025)

Microsoft AI Agents Hackathon Project

Project Overview

StudBud is an AI-powered study assistant that transforms how students engage with their learning materials. By integrating Java Spring Boot with Azure OpenAI and Azure SQL, it provides a seamless web interface for uploading study notes, generating intelligent summaries, creating interactive quizzes, and participating in contextual Q&A.

The application utilizes a modular agent architecture—incorporating specialized agents for tutoring, research, study planning, and quizzing—powered by LangChain4j for sophisticated, session-aware interactions.

My Contribution

  • Azure Foundry, Azure OpenAI, and Azure SQL setup
  • Implemented the agent architecture using LangChain4j, including the Tutor Agent, Research Agent, Study Planner Agent, and Quiz Agent.
  • Designed and developed the web interface using Java Spring Boot, enabling users to upload study materials, view summaries, take quizzes, and interact with the agents.
  • Implemented WebSocket-based real-time chat interface

Tech Stack

Java Spring Boot LangChain4j Azure Foundry Azure OpenAI Azure SQL
